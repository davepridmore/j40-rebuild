# Hose And Pipe Photo Capture Guide - 2026-05-04

Purpose: replace the placeholder/reference hose and pipe images with actual project photos that are good enough for ordering, fabrication, and later reassembly.

## How To Shoot

- Use daylight or flash. Wipe only enough grime to see fittings, clamps, markings, and hose printing.
- Put a ruler, tape, or caliper in every close-up. Put a small paper label in the frame when possible: `HLS-01`, `RPI-FUEL-002-A`, etc.
- For every circuit, take one wide route photo first, then close-ups at each end.
- Shoot square to the fitting or hose end. Avoid steep angles where diameter, thread, clamp land, or bend shape cannot be read.
- Do not remove hydraulic brake/clutch lines just for photos. If any line is removed later, cap/plug ports immediately and keep the old line as a bend template.

## Import

After taking the photos, select the batch in Google Photos and tell Codex to import them. Use the repository picker workflow:

```bash
./scripts/run_personal_photos_picker_on_demand.sh
```

That will import the selected images into `photos/`, rebuild `data/manual/photo_inventory.csv`, and refresh the project photo catalog.
