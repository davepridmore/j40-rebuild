# Bilal Ganj Mechanic Checklist

Date: 2026-04-15

Use with detailed sizes: `bilal-ganj-detailed-size-specs.md`

Priority reminder:
- Springs are included: rows `15` and `16` (measure then buy).

## 1) Tools to Carry

- Thread pitch gauges: M6/M8/M10/M12
- Vernier caliper
- Tape measure
- Marker + notebook
- Magnet and small wire brush
- One old body mount sample (rubber + sleeve)
- One reference bolt/nut sample from vehicle

## 2) Mandatory Checks by Item

### Fasteners (Rows 215, 244-247)

- Confirm pitch and diameter: M6x1.0, M8x1.25, M10x1.25, M12 where specified.
- Confirm property class markings on structural bolts: minimum `8.8`.
- Check thread quality by hand-run with matching nut (no binding).
- Reject mixed unmarked scrap bolts for structural mounts.
- Confirm yellow-zinc or corrosion-resistant finish for captive/rivnut sets.

### Body Mount Rubbers + Sleeves (Rows 12, 13)

- Compare against old sample:
  - Inner diameter (sleeve fit)
  - Outer diameter (cup/seat fit)
  - Stack height (affects tub alignment)
- Check rubber hardness consistency across set.
- Confirm full set count for all mount points before paying.

### Ignition Lock Security Fix (Row 41)

- Validate key operation through full positions (ACC/IGN/START if applicable).
- Confirm no excessive play in barrel.
- If re-key job: test all supplied keys on-site.
- Ensure steering lock function is positive if lock assembly includes it.

### Fuse Boxes (Row 186)

- Must be compact and covered (no open exposed layout).
- Confirm lid locks firmly and does not pop open.
- Check terminal studs/screws are tight and not stripped.
- Buy 3 identical units to simplify future service.

### EPS Kit (Row 229)

- Use full checklist in `eps-bilal-ganj-kit-checklist.md`.
- Ask specifically for a complete 2005-2011 Toyota Vitz/Yaris 90-series column-assist EPS set, chassis code SCP90 or NCP90.
- Do not accept Corolla/Axio/Prius EPS as a buy item, hydraulic power steering parts, loose motors, missing ECUs, missing shafts, or cut-flush connectors.
- Mandatory in-person checks:
  - Donor year/chassis confirmation: 2005-2011 SCP90 or NCP90.
  - Complete matched kit presence (column motor unit, matching ECU/control, shafts/U-joints, couplers, brackets, connectors, pigtails).
  - Bench assist smoothness (no jerks/grinding).
  - Input/output play check (no major backlash).
  - Confirm connector pigtails are present and at least 150mm long.
  - Photograph labels, splines, connectors, brackets, and bench-test video.

### Heat Plugs / Glow Plugs (Row 29)

- NEW ONLY. Do not accept used/refurbished plugs.
- Confirm exact engine-code thread/reach/voltage spec before purchase.
- This is not a Bilal Ganj donor-scout item; order from a new Toyota/diesel parts counter after confirmation.

### Local Leaf Springs (Rows 15,16)

- Measure and record:
  - Eye-to-eye length
  - Pack width
  - Main leaf thickness + pack count
  - Center pin position
  - Arch height at unloaded state
- Confirm bush inner diameter and eye width match existing shackles/pins.
- Ask for pair-matching (left/right) and warranty terms.

### Wire Rework (Non-purchase task)

- Rework overlength heavy feed wire into 3 segments:
  - battery -> breaker
  - breaker -> junction/ANL
  - ANL -> destination branch
- Crimp and heat-shrink each segment; verify continuity and voltage-drop.

### OME/EMU Quote Items (Rows 33,44,45,46,47)

- Do not commit purchase without:
  - part-number photo
  - packaging/stamping evidence
  - written quote
- Verify application match against your axle/spring setup.
- Collect final quoted landed cost and lead time.

## 3) Buy / No-Buy Gates

- Buy immediately:
  - Fastener kits, body-mount rubbers/shims, ignition security, 3 compact covered cabin fuse boxes.
- Buy only if measured fit confirmed:
  - Local front/rear leaf packs.
- Quote only today:
  - OME/EMU package items.
- EPS:
  - Buy only as complete verified kit; reject incomplete bundles.

## 4) Data to Capture Before Leaving

- Seller name and stall location
- Part numbers photographed
- Quantity bought / quoted
- Final quoted and negotiated price
- Lead time if out of stock
- Return/exchange terms
