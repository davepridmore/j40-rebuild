## 8) Wire Rework Instruction (Non-purchase Task)

User-required correction:
- Overlength heavy feed wire that was previously made as one full run should be split into **3 serviceable segments**.

Recommended split points:
1. Battery positive -> breaker
2. Breaker -> junction / ANL entry
3. ANL output -> destination branch (for example EPS feed)

Build rule:
- Crimp with correct heavy lugs, label each segment, heat-shrink seal, then continuity + voltage-drop test.
