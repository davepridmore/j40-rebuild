# Window Refurbishment Workstream

Created: 2026-05-17

## Scope

This workstream covers all window-related repair and refit work:

- front windshield/front window glass and seals
- front vent/quarter window assemblies
- side-window glass, channels, sliders, and latches
- rear hatch/window latch mechanisms where applicable
- window rubbers, run channels, felts, glazing seals, and weatherstrips

Chassis body-mount rubbers, sleeves, and shims remain in the `chassis_rubbers` workstream.

## Workflow

Use `data/manual/window_parts_refurbishment_workflow.csv` as the control sheet. The active gates are:

1. Tag every glass/frame assembly by side, position, and orientation.
2. Bag and label latches, pivots, sliders, fasteners, springs, clips, and shims by assembly.
3. Inspect glass edges, rusted metal channels/tabs, latch/pivot wear, and missing hardware.
4. Measure rubber/weatherstrip profiles, glass thickness, channel width, total length, and lip direction before buying.
5. Clean, rust-treat, refinish, and lubricate reusable metal/mechanism parts.
6. Bench-fit and water-test before release to final body refit.

## Buy Gate

Do not buy a full window-rubber/weatherstrip set until each window type has either:

- a tagged reusable original seal, or
- a measured old sample/profile and a trial-fitted supplier sample.

Exact model-specific kits can be considered, but they still need confirmation against glass thickness, channel profile, door/body type, and repaired painted openings.
