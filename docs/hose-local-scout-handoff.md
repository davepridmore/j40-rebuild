# Hose Local Scout Handoff - Superseded

This page is retained only as a compatibility link for older dashboard references.

The hose and pipe workstream is no longer a local-market scout task. The active supplier-facing specification is now:

- [Longman Pipe And Hose Order Spec - 2026-05-12](longman-pipe-hose-order-spec-20260512.md)
- [Longman pipe/hose order CSV](../data/manual/longman_pipe_hose_order_specs.csv)

Active position: send Longman Mills the exact line IDs, dimensions, material ratings, sample requirements, and reject rules from the Longman spec. Old hoses, hard lines, clips, clamps, fittings, hydraulic assemblies, and related samples are for length/shape/fitting/routing identification only, not parts to reuse or buy used.

Brake and clutch flex hoses remain safety-controlled: Longman should quote them only if they can supply complete certified crimped hydraulic assemblies copied from the old assembly or matched to the current vehicle fittings. Do not use generic rubber hose for brake or clutch hydraulics.
