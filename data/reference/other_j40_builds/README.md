# Other J40 Build References

Put reference photos from other J40 builds in this folder, either directly or grouped by subfolder.

For existing WhatsApp/workshop samples that already live elsewhere in the repo, add them to `data/manual/other_build_reference_media.csv` instead of copying duplicates here.

After adding images, rebuild the dashboard:

```sh
python3 scripts/build_project_control_ui.py
```

Supported image formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`.
